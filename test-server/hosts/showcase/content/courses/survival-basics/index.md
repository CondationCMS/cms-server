## Stay calm. Make good decisions.

Over one focused weekend, you learn how to set priorities in an outdoor emergency. Every skill is explained clearly and then put into practice outside.

## What you will learn

- Light a fire using simple tools
- Find and safely treat water
- Build a weatherproof emergency shelter
- Establish your position and organise help
- Apply essential outdoor first aid

No previous experience is required. You receive a complete packing list four weeks before the course.
